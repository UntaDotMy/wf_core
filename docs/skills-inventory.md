# Skills Inventory

`wf_core` ships these Windsurf/Devin-compatible skills. Each skill uses the
standard `SKILL.md` format, includes richer upstream-derived reference packs, and
is installed into both Windsurf global homes and Devin's native global skills
home.

| Skill | Purpose |
| --- | --- |
| `backend-and-data-architecture` | APIs, data models, caching, queues, and service boundaries |
| `cloud-and-devops-expert` | Infrastructure, CI/CD, containers, IaC, observability, rollout, and recovery |
| `git-expert` | Branching, commits, pull requests, merge conflicts, and safe recovery |
| `memory-status-reporter` | Durable rules, learning recaps, mistake ledgers, and user preference state |
| `mobile-development-life-cycle` | Android/iOS lifecycle, permissions, offline behavior, release, and testing |
| `preserve-existing-flow` | Owner-path tracing before changing existing behavior |
| `qa-and-automation-engineer` | Test strategy, E2E coverage, regression proof, and flake triage |
| `reviewer` | Bugs, regressions, security issues, missing tests, and production readiness |
| `security-and-compliance-auditor` | Threats, auth, secrets, data handling, dependency and compliance risks |
| `software-development-life-cycle` | Planning, sequencing, architecture framing, and closure proof |
| `ui-design-systems-and-responsive-interfaces` | Responsive UI, accessibility, component systems, and frontend quality |
| `ux-research-and-experience-strategy` | Research, journeys, decision architecture, usability evidence, and recovery paths |
| `web-development-life-cycle` | Web architecture, browser behavior, performance, SEO, deployment, and compatibility |
