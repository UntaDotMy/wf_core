# Security Source Anchors

Use these references to ground security and compliance guidance in authoritative or canonical sources.

## Application Security

- OWASP Top 10: https://owasp.org/www-project-top-ten/
- OWASP API Security Top 10 2023: https://owasp.org/API-Security/editions/2023/en/0x11-t10/
- OWASP ASVS: https://owasp.org/www-project-application-security-verification-standard/
- OWASP Secrets Management Cheat Sheet: https://cheatsheetseries.owasp.org/cheatsheets/Secrets_Management_Cheat_Sheet.html
- NIST Secure Software Development Framework (SP 800-218): https://csrc.nist.gov/publications/detail/sp/800-218/final
- MITRE CWE Top 25: https://cwe.mitre.org/top25/

## Infrastructure and Secrets

- Kubernetes Secrets: https://kubernetes.io/docs/concepts/configuration/secret/
- Kubernetes RBAC: https://kubernetes.io/docs/reference/access-authn-authz/rbac/
- Kubernetes application security checklist: https://kubernetes.io/docs/concepts/security/application-security-checklist/
- GitHub secret scanning overview: https://docs.github.com/github/administering-a-repository/about-token-scanning
- GitHub secret scanning detection scope: https://docs.github.com/en/code-security/reference/secret-security/secret-scanning-detection-scope

## Compliance and Governance

- GDPR official text: https://eur-lex.europa.eu/eli/reg/2016/679/oj/eng
- HIPAA Security Rule overview: https://www.hhs.gov/ocr/privacy/hipaa/administrative/securityrule/index.html
- HHS guidance on HIPAA risk analysis: https://www.hhs.gov/hipaa/for-professionals/security/guidance/guidance-risk-analysis/index.html
- AICPA SOC 2 overview: https://www.aicpa-cima.com/topic/audit-assurance/audit-and-assurance-greater-than-soc-2
