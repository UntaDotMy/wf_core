# Security and Compliance Auditor Knowledge Map

Use this map before detailed execution.

## Core Lanes

- Threat models, trust boundaries, and authz/authn
- Secret handling, dependency, and supply-chain risk
- Compliance evidence and remediation prioritization

## Evidence To Collect

- Current behavior or requirement source
- Files, commands, and owners touched by the task
- Validation needed and validation actually run
- Residual risks or explicit blockers
