# Reviewer Knowledge Map

Use this map before detailed execution.

## Core Lanes

- Correctness, regressions, and missing tests
- Security, secrets, data loss, and concurrency risk
- Production-readiness findings with precise evidence

## Evidence To Collect

- Current behavior or requirement source
- Files, commands, and owners touched by the task
- Validation needed and validation actually run
- Residual risks or explicit blockers
