# Web Development Life Cycle Knowledge Map

Use this map before detailed execution.

## Core Lanes

- Web architecture, routing, state, and browser behavior
- Performance, SEO, compatibility, and security headers
- Testing, deployment, monitoring, and rollback

## Evidence To Collect

- Current behavior or requirement source
- Files, commands, and owners touched by the task
- Validation needed and validation actually run
- Residual risks or explicit blockers
